I. Limites

  1. Quelques limites usuelles

    2. \[\forall n\in\mathbb{N}^{*} \lim_{x \to +\infty} x^{n}=+\infty \]

    \[\lim_{x \to +\infty} \frac{1}{x^{n}}=0^{+} \]

    \[ \forall n\in\mathbb{N}^{*} \lim_{x \to -\infty} x^{n} = \begin{cases} +\infty \text{ si \(n\) est pair}\\ -\infty \text{ si \(n\) est impair} \end{cases} \]

    \[ \lim_{x \to -\infty} \frac{1}{x^{n}}= \begin{cases} 0^{+} \text{ si \(n\) est pair}\\ 0^{-} \text{ si \(n\) est impair} \end{cases} \]

    \[ \lim_{x \to +\infty} \sqrt{x}=+\infty \]

    \[ \lim_{x \to +\infty} \frac{1}{\sqrt{x}}=0^{+} \]

    \[ \lim_{x \to a^{+}} \frac{1}{x-a}=+\infty \]

    \[ \lim_{x \to a^{-}} \frac{1}{x-a}=-\infty \]

    Autre Notation

    \[( x\rightarrow a^{+} ) \iff ( x\rightarrow a_{>} ) \]

    \[( x\rightarrow a^{-} ) \iff ( x\rightarrow a_{<} ) \]

    \[ \lim_{x \to a^{+}} f(x)=+\infty \text{ se note aussi } \lim_{x \to a_{>}} f(x)=+\infty \]

    \[ \lim_{x \to a^{-}} f(x)=-\infty \text{ se note aussi } \lim_{x \to a_{<}} f(x)=-\infty \]

  2. Quelques théorèmes sur les limites
    3. Théorème de comparaison

    Soit $f$ et $g$ deux fonctions telles que $f(x) \leq g(x)$ sur un intervalle $]a; +\infty[$ de $\mathbb{R}$.

    Cas 1 : Limite en $+\infty$
    \[ \lim_{x \to +\infty} f(x) = +\infty \Rightarrow \lim_{x \to +\infty} g(x) = +\infty. \]
    \[ \lim_{x \to +\infty} g(x) = -\infty \Rightarrow \lim_{x \to +\infty} f(x) = -\infty. \]
    Cas 2 : Limite en $-\infty$

    Soit $f$ et $g$ deux fonctions telles que $f(x) \leq g(x)$ sur un intervalle $]-\infty; \beta[$ de $\mathbb{R}$.

    \[ \lim_{x \to -\infty} f(x) = +\infty \Rightarrow \lim_{x \to -\infty} g(x) = +\infty. \]
    \[ \lim_{x \to -\infty} g(x) = -\infty \Rightarrow \lim_{x \to -\infty} f(x) = -\infty. \]
    Cas 3 : Limite en un point $x_0$

    Soit $f$ et $g$ deux fonctions telles que $f(x) \leq g(x)$ sur un intervalle $]a; \beta[$ de $\mathbb{R}$ et $x_0 \in ]a; \beta[$.

    \[ \lim_{x \to x_0} f(x) = +\infty \Rightarrow \lim_{x \to x_0} g(x) = +\infty. \]
    \[ \lim_{x \to x_0} g(x) = -\infty \Rightarrow \lim_{x \to x_0} f(x) = -\infty. \]
    Exemple 1
    \[ \text{Soit } f(x) = -x + \sin x \text{. Calculer } \lim_{x \to +\infty} f(x) \]
    Solution
    Étape 1 : Encadrement de la fonction
    \[ \text{Pour tout } x, \sin x \leq 1 \implies -x + \sin x \leq 1 - x \implies f(x) \leq 1 - x \]
    Étape 2 : Définition de la fonction de comparaison
    \[ \text{Posons } g(x) = 1 - x \text{, donc } f(x) \leq g(x) \]
    Étape 3 : Calcul de la limite de comparaison
    \[ \lim_{x \to +\infty} g(x) = \lim_{x \to +\infty} (1 - x) = -\infty \]
    Étape 4 : Application du théorème
    \[ \text{Or, } \lim_{x \to +\infty} g(x) = -\infty \text{, donc } \lim_{x \to +\infty} f(x) = -\infty \]
    Exemple 2
    \[ \text{Soit } g(x) = \frac{\sqrt{1 + x^{2}}}{x^{2}} \text{. Calculer } \lim_{x \to 0} g(x) \]
    Solution
    Étape 1 : Encadrement de la fonction
    \[ \text{Posons } f(x) = \frac{1}{x^{2}} \text{. Comme, pour tout } x \neq 0,\\ \text{ on a } 1 \leq \sqrt{1 + x^{2}} \]
    Étape 2 : Inégalité entre les fonctions
    \[ \text{Donc, pour tout } x \neq 0, f(x) \geq g(x) \]
    Étape 3 : Calcul de la limite de comparaison
    \[ \lim_{x \to 0} f(x) = \lim_{x \to 0} \frac{1}{x^{2}} = +\infty \]
    Étape 4 : Application du théorème
    \[ \text{Or, } \lim_{x \to 0} f(x) = +\infty \text{, donc } \lim_{x \to 0} g(x) = +\infty \]
